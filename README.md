# IA PulsePredict

This repository aims to provide tools for generating synthetic health data and building predictive models for health outcomes. 

## Installation

To install the necessary packages, run:

```
pip install -r requirements.txt
```

## Usage

To run the app, use:

```
streamlit run app/app.py
```

## Directory Structure

- `src/`: Source code for the project.
- `models/`: Model training and evaluation utilities.
- `data/`: Data storage for raw and processed data.
- `notebooks/`: Jupyter notebooks for exploration and EDA.
- `tests/`: Test cases for the project.
